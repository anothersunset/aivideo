# 《影狩罗刹帖》开发包目录说明

## 1. 开发包状态

本目录为原创成人向 2D 忍者动作动画电影《影狩罗刹帖》的开发包。

项目方向：硬派成人向忍者动作、黑暗奇幻、架空时代剧、瘟疫兵器阴谋。

创作原则：可对齐经典成人向忍者动画的高层气质，但角色、组织、剧情、能力、镜头和视觉表达必须原创。

当前版本：v0.1 开发包。

当前用途：
- 内部开发会。
- 融资初谈。
- 样片启动准备。
- 角色、美术、动作、摄影初步开发。

## 3. 文件分组

### A. 项目总览

《影狩罗刹帖_项目圣经_v0.1.md》

用途：项目母文件。包含定位、世界观、角色、敌人、三幕结构、视觉声音、制作路线、预算档位和风险。

《影狩罗刹帖_投资版一页纸_v0.1.md》

用途：对外快速介绍项目，用于投资人、平台、联合制作初谈。

### B. 剧本开发

《影狩罗刹帖_5页剧情大纲_v0.1.md》

用途：完整故事概览，供编剧、导演、投资方理解长片主线。

《影狩罗刹帖_剧本二稿开发备忘录_v0.1.md》

用途：指导剧本二稿开发，明确结构、角色弧线、动作戏功能和场景规模控制。

《影狩罗刹帖_主要角色小传_v0.1.md》

用途：服务编剧、配音、角色设计和表演方向。

### C. 敌人与视觉开发

《影狩罗刹帖_百骸八众视觉关键词表_v0.1.md》

用途：指导八名核心敌人的剪影、材质、动作规则和设计禁区。

《影狩罗刹帖_视觉参考板清单_v0.1.md》

用途：指导美术团队建立合法、原创、可执行的参考板。

### D. 样片开发

《影狩罗刹帖_试制片段_雨夜温泉宿_剧本_v0.1.md》

## 5. 使用规则

对外发送：
- 不发送内部视觉参考原图。
- 不公开绑定具体参照作品。
- 不发送未审查的高风险尺度描述。
- 投资人优先发送一页纸、PPT、预算排期摘要。

对内开发：
- 所有新增文件必须标版本号。
- 剧本、分镜、预算、角色设计变更必须同步总制片。
- 新增主要角色、大场景、S 级动作戏必须经导演和总制片共同批准。
