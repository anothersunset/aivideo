# 《影狩罗刹帖》开发会议议程与责任分工 v0.1

## 1. 文件目的

本文件用于把当前开发包转化为实际会议、决策和派工流程。项目已具备项目圣经、剧情大纲、角色小传、敌人设计方向、样片剧本、镜头表、预算排期、融资材料和尺度策略，下一步必须进入有节奏的开发管理。

管理原则：
- 每场会议只解决明确问题。
- 会前必须有输入文件，会后必须有输出决议。
- 创作判断由导演负责，预算和排期判断由总制片负责。
- 未经导演与总制片同时确认，不新增主要角色、大场景或大动作戏。

## 2. 核心决策组

| 角色 | 责任 | 最终确认范围 |
|---|---|---|
| 制作商/出品方 | 投资目标、商业边界、发行方向 | 融资阶段、预算上限、发行策略 |
| 导演 | 创作总判断、风格、表演、动作节奏 | 剧本方向、分镜、视觉基调、终剪 |
| 总制片 | 预算、排期、团队、风险控制 | 制作规模、现金流、交付节点 |
| 编剧 | 剧本结构、对白、人物弧线 | 剧本一稿、二稿、导演稿执行 |
| 美术监督 | 世界视觉、场景、色彩 | 美术设定、色彩脚本、场景风格 |
| 角色设计 | 角色与敌人造型 | 角色方向稿、三视图、表情表 |
| 动作导演 | 战斗规则、动作设计 | 动作戏拆解、关键 Pose、镜头难度 |
| 摄影监督 | 合成、光影、特效语言 | 摄影测试、雨烟火血毒膜方案 |
| 音响监督 | 声音风格、拟音、配乐方向 | 样片声音、配音风格、混音方向 |
| 法务/发行顾问 | 原创性、尺度、地区风险 | 法务审查、评级建议、国际版策略 |

## 3. 第一场：项目绿灯预备会

时长：90 分钟。

参会：
- 制作商
- 导演
- 总制片
- 编剧
- 美术监督
- 动作导演
- 发行/法务顾问

会前输入：
- 项目圣经 v0.1
- 投资版一页纸 v0.1
- 24 个月制片总排期 v0.1
- 预算与团队规划 v0.1

议程：
1. 确认项目定位：成人向 2D 忍者动作电影。
2. 确认原创边界：只对齐气质，不复刻参照作品。
3. 确认目标预算：中档 350-500 万美元。
4. 确认第一阶段目标：剧本二稿、视觉方向稿、试制片段。
5. 确认融资样片是否作为第一阶段核心交付。

会后输出：
- 项目是否进入开发阶段。
- 第一阶段预算上限。
- 样片是否启动。
- 需修改的项目定位条目。

## 4. 第二场：剧本二稿启动会

时长：120 分钟。

参会：
- 导演
- 编剧
- 总制片
- 动作导演
- 美术监督
- 法务/发行顾问

会前输入：
- 5 页剧情大纲 v0.1
- 主要角色小传 v0.1
- 剧本二稿开发备忘录 v0.1
- 暴力尺度与国际版剪辑策略 v0.1

议程：
1. 确认三幕结构和目标时长。
2. 确认凛藏、鸢、玄马斋、朽叶四条核心弧线。
3. 确认 9 场动作戏是否全部保留。
4. 标记高风险尺度场面。
5. 确认场景数量上限。
6. 确认二稿交付格式和时间。

会后输出：
- 剧本二稿任务书。
- 必须强化戏份清单。
- 必须删除或压缩的风险项。
- 二稿提交日期。

## 5. 第三场：角色与敌人设计启动会

时长：120 分钟。

参会：
- 导演
- 角色设计
- 美术监督
- 动作导演
- 作画监督
- 摄影监督
- 总制片

会前输入：
- 主要角色小传 v0.1
- 百骸八众视觉关键词表 v0.1
- 视觉参考板清单 v0.1

议程：
1. 确认凛藏、鸢、水母尼为第一批设计对象。
2. 确认百骸八众第一轮只做剪影，不进入复杂细节。
3. 确认可动画化原则：高速动作角色不能过度装饰。
4. 确认水母尼、朽叶为 S 级设计对象。
5. 确认敌人设计禁区和原创性风险。

会后输出：
- 第一批角色设计任务单。
- 百骸八众剪影交付日期。
- 样片角色设定优先级。
- 需法务关注的视觉风险点。

## 6. 第四场：样片分镜启动会

时长：120 分钟。

参会：
- 导演
- 分镜导演
- 动作导演
- 摄影监督
- 美术监督
- 音响监督
- 总制片

会前输入：
- 《雨夜温泉宿》试制片段剧本 v0.1
- 《雨夜温泉宿》试制片镜头表 v0.1
- 《雨夜温泉宿》分镜制作说明 v0.1
- 《雨夜温泉宿》试制片生产清单 v0.1

议程：
1. 确认 21 镜结构和 90-120 秒目标。
2. 确认 S 级镜头：007、008、012、015、018、019、021。
3. 确认温泉宿空间关系和平面图需求。
4. 确认动作清晰度原则。
5. 确认声音点和摄影测试点。
6. 确认分镜交付格式。

会后输出：
- 分镜任务书。
- 温泉宿平面图任务。
- S 级镜头负责人建议。
- 动态分镜交付时间。

## 7. 第五场：视觉与摄影测试会

时长：90 分钟。

参会：
- 导演
- 美术监督
- 摄影监督
- 特效负责人
- 角色设计
- 总制片

会前输入：
- 视觉参考板清单 v0.1
- 百骸八众视觉关键词表 v0.1
- 样片镜头表 v0.1

议程：
1. 确认雨、温泉蒸汽、幻觉香、药烟、毒膜五种效果差异。
2. 确认夜景不能黑死。
3. 确认毒膜不是玻璃滤镜，而是湿润活体薄膜。
4. 确认血色系统：普通血、黑热血、水母尼蓝血。
5. 确认第一批摄影测试帧。

会后输出：
- 摄影测试任务单。
- 色彩脚本初步方向。
- 样片特效风险清单。

## 8. 第六场：融资材料审查会

时长：90 分钟。

参会：
- 制作商
- 总制片
- 导演
- 宣发负责人
- 发行顾问
- 法务

会前输入：
- 投资版一页纸 v0.1
- 投资版 PPT 大纲 v0.1
- 预算与团队规划 v0.1
- 暴力尺度与国际版剪辑策略 v0.1

议程：
1. 确认对外表达是否避免绑定参照作品。
2. 确认融资需求和第一阶段金额。
3. 确认样片预算口径：8-15 万美元。
4. 确认目标会议对象：平台、联合制片、私募、发行。
5. 确认 PPT 所需视觉素材。

会后输出：
- 融资 PPT 制作任务。
- 投资人会议名单。
- 法务修改意见。
- 宣传禁用词清单。

## 9. 每周例会机制

频率：每周一次，60 分钟。

固定参会：
- 导演
- 总制片
- 制片主任
- 本周相关部门负责人

固定议程：
1. 上周交付物验收。
2. 本周阻塞项。
3. 预算和排期变化。
4. 创作变更请求。
5. 下周交付物确认。

例会输出：
- 决议记录。
- 任务表更新。
- 风险清单更新。
- 需要制作商确认的事项。

## 10. 变更审批规则

必须经过导演和总制片共同批准：
- 新增主要角色。
- 新增百骸八众之外的强敌。
- 新增大场景。
- 新增 S 级动作戏。
- 改变结局。
- 改变目标评级。
- 超出样片预算上限。

必须经过法务/发行顾问确认：
- 对外公开提及参照作品。
- 使用真实历史组织、家纹或宗教符号。
- 涉及高风险尺度镜头。
- 改变国际版策略。
