# 《影狩罗刹帖》开发阶段 4 周工作计划 v0.1

## 1. 计划目标

周期：4 周。

阶段目标：把当前开发包推进到“可开融资会、可启动样片分镜、可进入剧本二稿”的状态。

4 周结束时必须具备：
- 剧本二稿初版或详细场景大纲。
- 凛藏、鸢、水母尼角色方向稿。
- 百骸八众第一轮剪影。
- 《雨夜温泉宿》分镜 v0.1。
- 温泉宿平面图。
- 摄影测试帧方案。
- 投资 PPT v0.1。
- 样片预算细化表。

## 2. 工作流总览

| 周次 | 主题 | 核心目标 | 关键交付 |
|---|---|---|---|
| W1 | 锁方向 | 统一创作、预算、样片目标 | 会议决议、任务书、平面图需求 |
| W2 | 出草稿 | 剧本、角色、分镜、美术同步起稿 | 剧本场景大纲、角色方向、分镜草稿 |
| W3 | 动态验证 | 动态分镜、摄影测试、PPT 视觉化 | 动态分镜粗版、测试帧、PPT 初版 |
| W4 | 审查收束 | 统一修改，形成融资与样片开工包 | 开发包 v0.2、样片预算、审查意见 |

## 3. W1：锁方向

### 目标

让所有核心人员对项目定位、原创边界、样片目标和第一阶段成果达成一致。

### 会议

1. 项目绿灯预备会。
2. 剧本二稿启动会。
3. 角色与敌人设计启动会。
4. 样片分镜启动会。

### 任务

| 任务 | 主责 | 输出 |
|---|---|---|
| 确认项目定位和预算口径 | 制作商、总制片、导演 | 会议决议 |
| 剧本二稿任务拆解 | 编剧、导演 | 二稿任务表 |
| 凛藏/鸢/水母尼设计 brief | 角色设计、导演 | 角色设计任务书 |
| 温泉宿平面图需求 | 美术监督、分镜导演 | 平面图草案需求 |
| 样片 S 级镜头确认 | 导演、动作导演、总制片 | S 镜头责任表 |
| 融资 PPT 素材缺口确认 | 总制片、宣发、美术 | 素材缺口表 |

### W1 验收标准

- 所有主创同意“成人向 2D 忍者动作电影”定位。
- 确认不公开绑定参照作品。
- 样片预算口径锁定为 8-15 万美元。
- 剧本二稿、角色方向稿、样片分镜都有负责人和截止时间。

## 4. W2：出草稿

### 目标

让剧本、美术、角色、分镜四条线同时出第一批可讨论材料。

### 任务

| 任务 | 主责 | 输出 |
|---|---|---|
| 剧本二稿场景大纲 | 编剧 | 90-105 分钟场景大纲 |
| 第一幕关键场面扩写 | 编剧 | 山路追杀、温泉宿段落 |
| 凛藏方向稿 | 角色设计 | 3 个方向 |
| 鸢方向稿 | 角色设计 | 3 个方向 |
| 水母尼方向稿 | 角色设计 | 3 个方向 |
| 百骸八众剪影第一轮 | 角色设计 | 每人 3 个剪影 |
| 温泉宿平面图 | 美术监督 | 大堂、暗室、走廊、温泉池、屋檐 |
| 样片分镜草稿 | 分镜导演 | 21 镜草稿 |
| 投资 PPT 文字版 | 宣发/制片 | 14 页文字版 |

### W2 审查会

时长：120 分钟。

审查内容：
- 剧本场景数量是否失控。
- 角色方向是否可动画化。
- 百骸八众剪影是否足够区分。
- 温泉宿空间是否支持镜头表。
- 分镜草稿是否看得懂动作关系。

### W2 验收标准

- 凛藏、鸢、水母尼至少各选出 1 个方向进入细化。
- 百骸八众剪影不互相混淆。
- 温泉宿平面图支持 21 镜调度。
- 分镜草稿能剪成粗动态。

## 5. W3：动态验证

### 目标

用动态分镜和测试帧验证样片节奏、摄影难点和融资表达。

### 任务

| 任务 | 主责 | 输出 |
|---|---|---|
| 样片动态分镜粗版 | 分镜导演/剪辑 | 90-120 秒粗动态 |
| 临时音效点 | 音响监督 | 雨、刀、烟、毒膜、火 |
| 水母尼毒膜测试帧 | 摄影监督/特效 | 2-3 张测试 |
| 雨夜温泉宿色彩测试 | 美术监督/摄影 | 2 张测试 |
| 凛藏三连击 Pose | 动作导演/原画 | 关键 Pose |
| 鸢行动 Pose | 动作导演/角色设计 | 射灯、药烟、挡针 |
| 投资 PPT v0.1 | 宣发/制片 | 可演示版 |
| 样片预算细化 | 总制片/制片主任 | 按镜头预算表 |

### W3 审查会

时长：120 分钟。

审查内容：
- 动态分镜时长是否在 90-120 秒。
- S 级镜头是否过多或过难。
- 毒膜效果是否成立。
- 雨夜画面是否可读。
- PPT 是否能在 10-15 分钟讲清楚项目。

### W3 验收标准

- 动态分镜能完整看懂剧情和动作。
- 摄影测试明确可行方向。
- 样片预算没有明显漏项。
- 投资 PPT 可用于内部试讲。

## 6. W4：审查收束

### 目标

整合前三周反馈，形成开发包 v0.2，并决定是否启动样片正式制作。

### 任务

| 任务 | 主责 | 输出 |
|---|---|---|
| 剧本二稿初版或详细大纲修订 | 编剧 | 二稿材料 v0.1 |
| 样片分镜 v0.1 | 分镜导演 | 完整分镜 |
| 动态分镜 v0.1 | 剪辑/分镜导演 | 带临时声音版本 |
| 角色方向稿 v0.1 | 角色设计 | 凛藏、鸢、水母尼 |
| 百骸八众剪影 v0.1 | 角色设计 | 8 人剪影表 |
| 摄影测试帧 v0.1 | 摄影监督 | 雨、烟、毒膜测试 |
| 投资 PPT v0.2 | 宣发/制片 | 对外会议版 |
| 样片制作预算 v0.1 | 总制片 | 8 周预算与人力 |
| 法务原创性初审 | 法务 | 风险意见 |

### W4 终审会

时长：150 分钟。

参会：
- 制作商
- 导演
- 总制片
- 编剧
- 美术监督
- 角色设计
- 动作导演
- 摄影监督
- 宣发/发行
- 法务

决策事项：
1. 是否进入样片正式制作。
2. 是否批准剧本二稿继续完整扩写。
3. 是否批准投资 PPT 对外使用。
4. 是否需要调整预算档位。
5. 是否存在必须立即规避的原创性或尺度风险。

### W4 验收标准

- 样片制作范围明确。
- 样片预算明确。
- 样片核心人员明确。
- 投资 PPT 可对外。
- 剧本二稿下一步明确。
- 法务风险无红灯项。

## 7. 每周固定交付格式

每周五提交：
- 本周完成文件。
- 未完成任务说明。
- 风险清单。
- 下周任务。
- 需要制作商确认的问题。

文件命名：
- 项目名_内容_版本_日期
- 示例：影狩罗刹帖_样片动态分镜_v0.1_2026xxxx

版本规则：
- v0.1：首次可讨论版。
- v0.2：内部审查修订版。
- v0.9：待锁定版。
- v1.0：锁定版。

## 8. 风险监控

每周必须检查：
1. 剧本是否新增大场景。
2. 动作戏是否超出 6 场核心战斗策略。
3. 角色设计是否过度复杂。
4. 敌人是否过度接近已有作品。
5. 样片是否超出 120 秒。
6. 摄影测试是否不可控。
7. 投资 PPT 是否误用参照作品作为卖点。
8. 样片预算是否超过上限。

## 9. 第一阶段完成定义

4 周后若满足以下条件，即可视为第一阶段开发可进入样片制作：
- 剧本二稿方向明确。
- 样片分镜和动态分镜通过。
- 样片预算通过。
- 凛藏、鸢、水母尼方向稿通过。
- 温泉宿场景设计可生产。
- 摄影测试方向通过。
- 投资 PPT 可对外试讲。
- 法务原创性无重大风险。
