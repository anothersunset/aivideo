# 《影狩罗刹帖》雨夜温泉宿试制片生产清单 v0.1

## 1. 样片目标

片段：雨夜温泉宿

目标时长：90-120 秒，当前镜头表版本约 112 秒。

用途：
- 融资演示。
- 核心团队招募。
- 平台/发行销售材料。
- 动作、美术、摄影、声音生产测试。

样片必须证明：
1. 本项目成人向忍者动作气质成立。
2. 凛藏、鸢、水母尼三人的角色关系成立。
3. 雨、烟、血、火、透明毒膜的视觉语言可生产。
4. 有足够强的国际类型片辨识度。
5. 不依赖参照作品具体元素，保持原创。

## 2. 样片生产周期

推荐周期：8 周。

压缩周期：6 周，仅适用于核心人员全职到位。

理想周期：10 周，适用于需要高完成度融资展示。

## 5. 镜头优先级

必须优先完成的 S 级镜头：
- 007：凛藏破屋顶坠入。
- 008：凛藏三连击。
- 012：刀斩毒膜被滑开。
- 015：薄丝缠刀与短刀被夹。
- 018：白幕剪影战斗。
- 019：灯笼落水，水母尼毒膜显形。
- 021：雨中结尾与手腕烧痕。

原因：
- 这些镜头决定样片能否卖出项目气质。
- 这些镜头可测试全片最关键的动作和摄影方案。
- 若预算不足，其他镜头可降规格，S 级镜头不能降。

## 6. 预算估算

推荐样片预算：8-15 万美元。

低配可行预算：5-8 万美元。

高完成度预算：15-25 万美元。

| 模块 | 低配 | 推荐 | 高完成度 |
|---|---:|---:|---:|
| 分镜与动态分镜 | 0.8-1.2 万 | 1.5-2 万 | 2-3 万 |
| 角色与场景设定 | 0.8-1.5 万 | 1.5-2.5 万 | 3-4 万 |
| 原画与作监 | 1.5-2.5 万 | 3-5 万 | 6-8 万 |
| 动画与上色 | 0.8-1.5 万 | 1.5-2.5 万 | 3-4 万 |
| 背景 | 0.5-1 万 | 1-1.8 万 | 2-3 万 |
| 摄影合成与特效 | 0.8-1.5 万 | 1.5-3 万 | 3-5 万 |
| 声音与音乐 | 0.5-0.8 万 | 0.8-1.5 万 | 2-3 万 |
| 制片与预备金 | 0.3-0.8 万 | 1-1.7 万 | 2-3 万 |

预算原则：
1. 优先给 S 级镜头。
2. 声音不能完全省，雨夜动作片段没有声音会显著降级。
3. 摄影合成必须参与早期测试，不能等画完再补救。
4. 低配版本可以减少群众和背景复杂度，但不能减少鸢的行动镜头。

## 7. 审查节点

### 分镜审查

通过标准：
- 21 镜动作关系清楚。
- 凛藏、鸢、水母尼都有明确存在感。
- 观众能理解“旧药”是主线钩子。
- 不依赖解释性对白。

### 动态分镜审查

通过标准：
- 总时长在 90-120 秒。
- 动作爆发点清楚。
- 雨、白幕、火光、毒膜显形有节奏变化。
- 结尾能引出长片故事。

### 原画一检

通过标准：
- S 级镜头 Pose 有力量。
- 凛藏动作短促、准确。
- 水母尼动作柔软异常。
- 鸢不是背景角色。

### 摄影测试审查

通过标准：
- 夜景可读。
- 三种雾效可区分。
- 毒膜不廉价。
- 血、水、火不会糊成一团。

### 最终样片审查

通过标准：
- 作为融资样片有冲击力。
- 作为生产测试能暴露真实成本。
- 作为创作样片能代表长片方向。
- 作为原创项目不显得依附参照作品。

## 8. 版本交付

内部审查版：
- 1920x1080 ProRes 或高码率 MP4。
- 带时间码。
- 带镜号水印。

融资展示版：
- 1920x1080 或 4K。
- 无镜号。
- 中英文片名卡。
- 结尾保留项目 LOGO 与联系方式占位。

制作归档：
- 分镜 PDF。
- 动态分镜工程。
- 角色设定。
- 背景设定。
- 分层工程文件。
- 音频分轨。
- 最终输出文件。

## 9. 立即开工条件

必须在开工前完成：
1. 镜头表 v0.1 通过导演与制片确认。
2. 角色方向稿至少完成凛藏、鸢、水母尼。
3. 温泉宿大堂平面图完成。
4. S 级镜头负责人确认。
5. 样片预算上限确认。
6. 文件命名、版本管理和审查节奏确认。

建议同时准备：
1. 投资版一页纸。
2. 视觉参考板。
3. 声音参考清单。
4. 暴力尺度参考。
5. 样片完成后的发行/融资会议名单。
